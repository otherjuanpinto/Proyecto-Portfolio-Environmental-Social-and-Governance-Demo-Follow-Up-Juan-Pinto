# Project 02 – Carbon Footprint of Coffee

## 1. Context
Carbon accounting is a core requirement under CSRD and climate strategies.

## 2. Objective
Estimate Scope 1, 2 and 3 emissions across the green coffee value chain.

## 3. Scope
- Sector: Green coffee
- Geography: Production → Export → Roasting
- Time horizon: Full lifecycle assessment

## 4. Data Sources
- FAOSTAT
- Our World in Data
- Shipping emissions datasets

## 5. Analytical Approach
GHG Protocol-based modeling and scenario analysis.

## 6. Outputs
- Emissions model
- Dashboards
- Reduction scenarios

## 7. Value Generated
Supports decarbonization strategies and regulatory compliance.
