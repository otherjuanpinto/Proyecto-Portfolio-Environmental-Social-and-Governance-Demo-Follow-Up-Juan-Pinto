# Project 06 – Sustainable Logistics Optimization

## 1. Context
Logistics costs and emissions are tightly linked in global coffee trade.

## 2. Objective
Optimize export routes considering cost and emissions trade-offs.

## 3. Scope
- Sector: Green coffee trade
- Geography: Latin America → Europe
- Optimization criteria: Cost, carbon, time

## 4. Data Sources
- UN Comtrade
- Shipping emissions datasets
- Port data

## 5. Analytical Approach
Optimization modeling, route analysis, scenario comparison.

## 6. Outputs
- Optimization model
- Scenario comparison
- Route recommendations

## 7. Value Generated
Improves operational efficiency while reducing carbon impact.
